# Manual de Uso - ItaGestão Board

## Acesso ao Sistema

1. Acesse a URL do sistema no navegador
2. Informe seu e-mail e senha
3. Clique em "Entrar"

No primeiro acesso, o sistema solicitará a alteração da senha.

## Dashboard

A tela inicial apresenta:
- Cartões com totais (abertas, concluídas, atrasadas, vencendo hoje, na semana)
- Gráfico de demandas por secretaria
- Gráfico de demandas por prioridade
- Últimas movimentações
- Ações rápidas

## Quadros

### Visualizar Quadros
- Menu lateral: "Quadros"
- Cada quadro exibe: título, secretaria, total de cartões e atrasados

### Criar Quadro
1. Clique em "Novo Quadro"
2. Preencha: título, descrição, secretaria, cor, tipo de fluxo e visibilidade
3. Clique em "Criar Quadro"
4. As listas serão criadas automaticamente conforme o tipo de fluxo

### Tipos de Fluxo
- **Padrão:** Para demandas gerais
- **Licitação:** Fluxo completo de processo licitatório
- **Obras:** Fluxo de obras e infraestrutura
- **Convênios:** Fluxo de convênios e TransfereGov

## Cartões (Tarefas)

### Criar Cartão
1. Dentro de um quadro, clique em "Novo Cartão"
2. Preencha: título, lista, descrição, secretaria, responsável, prioridade, prazo
3. Clique em "Criar Cartão"

### Mover Cartão
- Arraste o cartão de uma lista para outra (drag-and-drop)
- A movimentação é registrada automaticamente no histórico

### Detalhes do Cartão
Clique em um cartão para ver:
- Descrição completa
- Comentários
- Checklist
- Anexos
- Histórico de movimentações
- Informações laterais (responsável, prazo, prioridade, etc.)

### Comentários
- Digite o comentário no campo e clique "Enviar"
- Todos os comentários ficam registrados com data e autor

### Checklist
- Adicione itens à checklist
- Marque/desmarque conforme conclusão
- Útil para acompanhar sub-tarefas

### Anexos
- Clique em "Upload" para enviar arquivos
- Formatos aceitos: PDF, DOC, DOCX, XLS, XLSX, JPG, PNG, GIF, ZIP, RAR
- Tamanho máximo: 10MB por arquivo

## Minhas Tarefas

- Menu lateral: "Minhas Tarefas"
- Lista todas as tarefas atribuídas ao usuário logado
- Ordenadas por prazo (mais urgentes primeiro)

## Tarefas Atrasadas

- Menu lateral: "Tarefas Atrasadas"
- Exibe tarefas com prazo vencido
- Mostra quantidade de dias de atraso

## Tarefas do Dia

- Menu lateral: "Tarefas do Dia"
- Exibe tarefas com prazo para hoje

## Relatórios

- Menu lateral: "Relatórios"
- Filtros disponíveis: secretaria, prioridade, status, período
- Exportação em CSV (compatível com Excel)

## Gestão de Usuários

### Criar Usuário
1. Menu: "Usuários"
2. Clique em "Novo Usuário"
3. Preencha: nome, e-mail, CPF, telefone, perfil, secretaria, setor
4. Defina uma senha temporária
5. O usuário deverá alterar no primeiro acesso

### Perfis de Acesso

| Perfil | Permissões |
|--------|-----------|
| Administrador | Acesso total |
| Prefeito/Gabinete | Visualiza tudo, relatórios |
| Secretário | Gerencia sua secretaria |
| Coordenador | Cria/move tarefas do setor |
| Servidor | Executa tarefas atribuídas |
| Controle Interno | Visualiza histórico e prazos |
| Visualizador | Apenas consulta |

## Secretarias e Setores

- Menu: "Secretarias" para gerenciar secretarias
- Menu: "Setores" para gerenciar setores vinculados às secretarias

## Configurações

- Menu: "Configurações" (apenas administradores)
- Permite alterar configurações gerais do sistema

## Dicas de Uso

1. **Prioridades:** Use as prioridades (Urgente, Alta, Média, Baixa) para organizar o trabalho
2. **Prazos:** Defina prazos realistas para acompanhar atrasos
3. **Comentários:** Use comentários para registrar andamentos e decisões
4. **Checklists:** Divida tarefas complexas em sub-itens
5. **Anexos:** Anexe documentos relevantes ao cartão
6. **Etiquetas:** Use etiquetas coloridas para categorizar visualmente

## Suporte

Em caso de dúvidas ou problemas, entre em contato com o setor de TI da Prefeitura.
